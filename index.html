Lists

About

A list is a data structure.

It is similar to an array, except it is located in heap memory. This allows it to have an unfixied size, and expand at runtime. The prorgam typically doesn't index the array, or keep track of where elements are located. Instead, it uses a list object's methods to add and remove elements.


Methods

  • List.add(item)

    Appends the specified item to the end of a list.

  • List.add(index, item)

    Inserts the specified item at the specified position index in a list.

  • List.clear()

    Removes all of the elements from this list.

  • List.lastIndexOf(item)

    Returns the index in this list of the last occurrence of the specified element, or -1 if the list does not contain this element.

  • List.remove(index)

    Removes the element at the specified position in this list. Shifts any subsequent elements to the left (subtracts one from their indices).

  • List.get(index)

    Gets the element of a specified position within the list.

  • List.size()

    Returns the number of elements in this list i.e the size of the list.

  • List.contains(item)

    Returns true if this list contains the specified element.

  • List.indexOf(item)

    Returns the index in this list of the first occurrence of the specified element, or -1 if the list does not contain this element.


Usage Example

// Pseudocode.
list myList
myList.add(4)
myList.add(0, 2) // [2, 4]
myList.remove(1) // [2]
